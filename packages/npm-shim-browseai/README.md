# browseai

> **This package has been renamed to [`browseai-dev`](https://www.npmjs.com/package/browseai-dev).**

Installing `browseai` will automatically install `browseai-dev`.

## Install the main package directly

```bash
npx browseai-dev
```

Or install globally:

```bash
npm install -g browseai-dev
```

## What is BrowseAI Dev?

Reliable research infrastructure for AI agents — evidence-backed web search with confidence scores and citations.

- **MCP Server:** `npx browseai-dev` (works with Claude Desktop, Cursor, Windsurf)
- **REST API:** `https://browseai.dev/api`
- **Python SDK:** `pip install browseaidev`
- **LangChain:** `pip install langchain-browseaidev`
- **CrewAI:** `pip install crewai-browseaidev`
- **LlamaIndex:** `pip install llamaindex-browseaidev`
